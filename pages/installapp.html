VRMeeting - Android App - Return to Installation Guide

NOTE: The VRMeeting App requires an Android Device running Android 5.0 or later

Note: Your device must support and have installed the Google LLC Cardboard app from the playstore. If your device does not support this app then it will not able to run VRMeeting

Step 1: Enabling installation of 3rd party apps on your device

You'll need to allow the insallation of 3rd party apps on your device as VRMeeting is not hosted on the app store

To do this you can follow this guide

Step 2: Downloading the APK

Next you need to get the APK for the VRMeeting App onto your phone. You can do this in one of two ways:

Transfer via USB

You can transfer the APK to your device using a USB cable and enabling file transfer on your device. Then you just need to drag and drop the vrmeeting.apk from your downloaded distribution to a folder on your device

Download from the Web Server

If you have the Web Server installed then you can download the APK from your WebServer using the following URL:

Your-Web-Server-URL/download/apk

If you enter the URL into a browser on your device then it will download the APK. The APK is ~90MB

Step 3: Installing the App

If you followed Step 3 then you should now have the vrmeeting.apk file on your device. You'll want to select it and the option to install the APK will come up

Step 4: Configuring the App

Once the app is installed you should find the Hexcore logo in your apps

Pressing it will open the VRMeeting app to the homescreen.

You'll need to allow the app permission to access the file system and record audio. Otherwise it will not function correctly.

You won't be able to create an account or login until you setup the location of the Web Server

To configure the app, you'll need to click on the Settings button in the bottom left of the screen

You need to enter the domain name or IP Address of the machine hosting your Web Server and the port that it is listening on, by default this is 25560 and then click apply

Step : Finished!

The installation should now be complete. You should now be able to create an account and login

If you have any problems installing the Host Server feel free to Contact us.
