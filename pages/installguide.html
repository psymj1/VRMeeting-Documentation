VRMeeting - Installation Guide - Return to Index

The VRMeeting System is made up of 4 Sub-Systems. Below you'll find guides on how to install each of the systems.

If you have any problems installing any of the systems feel free to Contact us.

Before following the installation guides you'll need to pull down the latest distributable version of VRMeeting from The Distribution Repository

This contains all of the sub-systems precompiled and ready to run. This is ensures all of the sub-systems are compatible and tested

Each Installation Guide will then guide you through installing each sub-system of VRMeeting

Please Note: The Web Server will only run on a Windows Machine
